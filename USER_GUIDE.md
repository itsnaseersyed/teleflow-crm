## Getting Started

### What is TeleFlow CRM?

TeleFlow CRM is a **lead management and call tracking system** designed specifically for telecalling teams. It helps you:

- 📞 Manage a queue of customer leads to call
- 📊 Track call outcomes and status
- 📅 Schedule follow-ups efficiently
- 📈 View your performance metrics
- 🔄 Auto-advance to the next lead (save time!)

### System Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Stable internet connection
- Mobile phone for making calls
- Valid TeleFlow CRM login credentials

### First Time Setup

**Your admin will:**
1. Create your user account with email and password
2. Send you login credentials
3. Assign you your first batch of leads
4. You're ready to go!

## Logging In

### Step-by-Step Login Process

1. **Open the App**
   - Go to your company's TeleFlow CRM URL
   - You'll see the login page

2. **Enter Your Email**
   - Input the email your admin provided
   - Example: `yourname@company.com`

3. **Enter Your Password**
   - Use the password sent by your admin
   - Note: Save it securely

4. **Click "Sign In"**
   - If credentials are correct, you'll be logged in
   - Redirected to your dashboard

### Forgot Password?

- Contact your **admin** - they can reset your password
- Don't share your credentials with anyone

### Session Timeout

- For security, you'll be logged out after inactivity
- Log back in with your credentials
- All your work is saved

## Making Calls

### Before You Call

1. **Review Customer Details**
   - Read the name and phone number
   - Check the priority flag
   - Review any remarks about them

2. **Note Previous Call History** (if exists)
   - Previous call status
   - Last notes from last interaction
   - Follow-up dates scheduled

3. **Prepare Your Notes**
   - Have a notebook ready
   - Pen for writing quick notes
   - Service details if needed

### How to Make the Call

#### Option 1: Click Phone Number (Auto-Dial)
- Click the phone number directly
- Your phone's dialer opens
- Dial without typing

#### Option 2: Manual Dialing
- Note the phone number
- Open your phone
- Manually dial the number

#### Option 3: VoIP Integration (if available)
- Built-in call button makes call through app
- Hands-free options available

### During the Call

```
Call Duration Tips:
├─ Average call: 2-5 minutes
├─ Listen actively: Understand their needs
├─ Take notes: Write down key points
├─ Be professional: Polite and clear
└─ Handle objections: Address concerns calmly
```

### If Calls Fail

**Can't reach the customer?**
- Try again after 5 minutes
- Check if number is correct
- Note in feedback: "No response"
- Move to next lead

**Wrong number?**
- Verify the phone number
- Mark as "Invalid Number"
- Contact admin to check database

**Customer asks to call later?**
- Select "Busy" or "Follow-Up Needed"
- Schedule callback for later
- System will remind you

### Call Status Options Explained

#### ✅ Interested
- **Use when**: Customer shows genuine interest
- **Next step**: Follow-up call or conversion
- **Example**: "Yes, I'm interested in broadband plans"
- **Lead stays in queue** for follow-up

#### 📅 Follow-Up Needed
- **Use when**: Customer wants to discuss later
- **Next step**: Schedule specific callback date/time
- **Example**: "Call me back next week when I have time"
- **You'll see this** in "Follow-Ups" page on scheduled date

#### ❌ Not Interested
- **Use when**: Customer explicitly refuses
- **Next step**: Lead removed from active queue
- **Example**: "I'm not interested in this service"
- **Lead marked** as "Not Interested"

#### 🔕 Busy
- **Use when**: Customer is currently busy
- **Next step**: Retry in short time (24-48 hours)
- **Example**: "I'm in a meeting, will call you later"
- **Lead temporarily** on hold status

#### 📵 No Response
- **Use when**: Customer didn't pick up
- **Next step**: Voicemail left, or no answer
- **Example**: Call went to voicemail
- **Lead stays** for retry

#### 📴 Switched Off
- **Use when**: Number unreachable (switched off/no service)
- **Next step**: Contact admin to verify number
- **Example**: "The number cannot be reached"
- **Lead flagged** for verification

#### 🎉 Converted
- **Use when**: Customer made a purchase/enrollment
- **Next step**: You've successfully sold!
- **Example**: Customer buys internet plan
- **Lead moved to** "Converted" - GREAT JOB! 🏆

#### ⚠️ Invalid Number
- **Use when**: Number is incorrect/doesn't exist
- **Next step**: Report to admin for database fix
- **Example**: "This number doesn't exist"
- **Lead flagged** for admin review

### Filling in Feedback Notes

**Optional but Highly Recommended** - Notes help:

```
What to include:
├─ Customer's main concern
├─ Their specific needs
├─ Any promises you made
├─ Best time to call back
├─ Personal details (name pronunciation, etc.)
└─ Next action items

Good Note Example:
"Rajesh is interested in 100 Mbps plan. 
Wife needs to approve. Call back Friday 6PM 
after office. Speak slowly - hearing issue."

Bad Note Example:
"Busy" or "Will call later"
(Not specific enough for next call)
```

### Setting Follow-Up Dates

If you select **"📅 Follow-Up Needed"**:

1. **Pick a Date**
   - Click on calendar widget
   - Select the callback date
   - System shows available dates

2. **Note the Time** (optional)
   - Add time if customer specified
   - Write in feedback notes: "Callback at 2PM"

3. **System Reminder**
   - On that date, lead appears in **"Follow-Ups"** page
   - You'll be reminded to call back
   - Continue workflow from where you left off

### Saving Your Entry

**"Save & Next Lead"** (Primary Action)
- Saves your call data
- **Auto-advances to the next lead instantly**
- No page refresh needed
- Maximizes your productivity 🚀

**"Save"** (Save Only)
- Saves without auto-advance
- You choose when to continue
- Useful if you need a break

**"Cancel"**
- Discards unsaved changes
- Returns to lead without saving
- Use if you called wrong number

### Making Follow-Up Calls

1. **Check Today's Follow-Ups First**
2. **Review Previous Notes**
   - What was the last conversation?
   - What did you promise?
   - What's the customer's concern?

3. **Call the Customer**
   - Click phone number to dial
   - Be friendly and reference previous call
   - "Hi Rajesh, this is [Your Name] from TeleFlow..."

4. **Log the Outcome**
   - Same as regular call workflow
   - Select status
   - Add new notes if conversation changed
   - Save & Next

### Best Practices for Follow-Ups

✅ **DO:**
- Call on or before the scheduled date
- Start with: "We spoke about X..."
- Reference previous notes
- Be professional and timely

❌ **DON'T:**
- Miss scheduled follow-up dates
- Call at wrong time
- Forget what was discussed
- Be pushy if they still need time

## Tips & Best Practices

### Productivity Tips

#### 1. Work Through Your Queue Systematically
```
Morning Routine:
1. Log in early
2. Check today's follow-ups first
3. Start from "Lead 1 of X"
4. Work through continuously
5. Use "Save & Next" for speed
6. Take lunch after first 25 leads
```

#### 2. Master the "Save & Next" Feature
- **Fastest way** to move through leads
- Don't use Previous/Next buttons
- Auto-advance saves 5-10 seconds per lead
- Over a day: saves 30-60 minutes! ⏱️

#### 3. Use Notes Effectively
- **First call**: Write detailed notes
- **Follow-up call**: Read old notes
- Include: Needs, objections, promises
- Help your team if lead reassigned

#### 4. Manage Call Quality vs Quantity
```
Balance:
├─ Don't rush (bad customer experience)
├─ Don't spend 30min per call (low productivity)
├─ Sweet spot: 3-5 minutes per call
└─ Quality builds repeat business
```

### Call Handling Tips

#### For Receptionist/Gatekeeper
- **Be polite**: They control access
- **Ask**: "May I speak with [Customer]?"
- **Don't be pushy**: Try again later
- **Note status**: "Receptionist says call back Tuesday"

#### For Interested Customers
- **Listen actively**: What's their main need?
- **Clarify benefits**: How does your service help?
- **Ask for decision**: "Can you commit today?"
- **Schedule follow-up**: If they need time

#### For Objection Handling
- **Listen fully**: Understand their concern
- **Acknowledge**: "I understand your concern..."
- **Address**: Provide relevant information
- **Try again**: Offer different angle or timing

#### For "Not Interested"
- **Respect decision**: Don't push
- **Get reason**: "May I ask why?" (optional)
- **Leave door open**: "Call us if plans change"
- **Log accurately**: Mark as "Not Interested"

### Time Management

| Task | Time | Tips |
|------|------|------|
| **Per Call** | 3-5 min | Include dialing + talking |
| **Per Lead Entry** | 1 min | Quick status + note |
| **Per 50 Leads** | 3-4 hours | With breaks |
| **Follow-ups** | Priority | Scheduled customers first |

### Handling Difficult Situations

#### Customer is Rude
- Stay calm and professional
- Don't take it personally
- If abusive → "I'll transfer you to manager"
- Log: "Rude, not interested" + move on

#### Can't Reach Customer
- Try 2-3 times in same session
- If no response → Mark "No Response"
- Don't keep retrying same call
- Move to next lead in queue

#### Technical Issue During Call
- Write down customer details
- Explain: "I'll call you back in 2 minutes"
- Log partial info
- Resolve (refresh, restart) and callback

#### Lead Information is Wrong
- Verify with customer if possible
- If confirmed wrong → Mark "Invalid Number"
- Contact admin with details
- Use correct info if customer provides

## Troubleshooting

### I Can't Log In

**Problem**: Error message or page won't load

**Solutions:**
1. Check internet connection
2. Clear browser cache (Ctrl+Shift+Delete)
3. Try different browser
4. Verify email is correct
5. Check CAPS LOCK isn't on
6. **Contact admin to reset password**

### My Leads Aren't Showing

**Problem**: "No leads assigned" or empty queue

**Solutions:**
1. Refresh page (F5 or Ctrl+R)
2. Wait 30 seconds (real-time sync takes time)
3. Check if admin assigned leads
4. Try logging out and back in
5. **Contact admin** - They haven't assigned leads yet

### Phone Number Won't Dial

**Problem**: Click phone number but nothing happens

**Solutions:**
1. Your phone settings may block clicking
2. Manually copy/paste number to phone
3. Use different browser if available
4. Try Wi-Fi instead of mobile data
5. **Some older phones don't support auto-dial**

### Can't Save Call Entry

**Problem**: "Save & Next" button won't work

**Solutions:**
1. Check internet connection (critical!)
2. Make sure Call Status is selected (required field)
3. Close popup notifications blocking button
4. Try "Save" instead of "Save & Next"
5. Wait 5 seconds before trying again
6. **Refresh and try again** if persistent

### Follow-Up Dates Not Showing

**Problem**: Can't see or select follow-up date

**Solutions:**
1. Scroll down to see Calendar widget
2. Make sure you selected "Follow-Up Needed" status
3. Try clicking date field again
4. Use mobile browser if desktop fails
5. **Check browser date/time is correct**

### Call Status Got Wrong - Can I Fix It?

**Problem**: Saved wrong status for a lead

**Solutions:**
1. **Go back to lead** (find in queue)
2. **Click to edit** (if edit option available)
3. **Update the status**
4. **Save again**
5. Or **Contact admin** to fix in database

### Performance is Slow

**Problem**: App loads slowly, buttons lag

**Solutions:**
1. Check your internet speed (needs 5+ Mbps)
2. Close other browser tabs (save memory)
3. Refresh page (F5)
4. Clear browser cache
5. Use modern browser (Chrome/Firefox)
6. Try at different time (peak hours?)

### I'm Assigned While Another User Was on That Lead

**Problem**: Two people trying to work same lead

**Solutions:**
1. System **prevents this** with lead locking
2. You'll see "locked" indicator if someone else has it
3. You'll be shown the next available lead
4. Wait a few minutes and try again
5. Contact admin if it persists

### Still Need Help?

- What you were trying to do

- What you already tried
